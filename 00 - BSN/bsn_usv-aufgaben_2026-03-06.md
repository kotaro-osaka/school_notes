## 3. Warum kann man einen Server nicht direkt mit einer Batterie/Akku betreiben?
=> Server brauchen AC (Wechselstrom), Akkus liefern DC (Gleichstrom)
=> Begrenzte Kapazität (nicht für Dauerbetrieb geeignet)
