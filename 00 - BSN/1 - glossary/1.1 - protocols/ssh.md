# SSH
___
> Secure Shell
> Port: `22`
> TCP
> Usage: `Server administration`, `remote access`, `secure file transfer`
- Secure host-to-host communication over insecure network
- Securely log into remote system
## Functions
___
### SFTP
> SSH File Transfer Protocol
> TCP
- Secure FTP
### SCP
> Secure Copy Protocol
> TCP
> Usage: `File transfer between local host/remote host & remote host`
- Encrypts authentication & data during transmission with SSH
- Users authentication with SSH key or password
- Files or directories

- Port forwarding
- Password-Based Authentication
- Public key Authentication
- Host-Based Authentication
- Keyboard-Interactive Authentication
- Certificate-Based Authentication
